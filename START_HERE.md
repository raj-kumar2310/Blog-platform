## 🚀 Getting Started (5 Minutes)

### Step 1: Setup Backend
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your MongoDB URI
npm run dev
```

### Step 2: Setup Frontend (new terminal)
```bash
cd frontend
npm install
cp .env.example .env.local
npm run dev
```

### Step 3: Open Browser
```
http://localhost:3000
```

**That's it!** Your blog platform is running! 🎉
